== Changelog (Registro de cambios)

NOTE: le Changelog que retracia modificationes que occurreva ante le 27/09/2014 es
https://slint.fr/pub/slint-history/Changelog.txt[ci].

[format="psv", options="header,autowidth"]
|========
|Data|Modificationes
|27/09/2014|
On propone al usator crear usatores regular al fin de le installation. +
Imagines "Full" ISO (continente omne pacchettos expedite con le imagines DVD ISO de  Slackware  in addition al installator Slint ) es disponibile http://sourceforge.net/projects/slint/files/[ci] +
Nederlandese nunc es disponibile, gratias a Dik Leenheer e Eric Hameleers. +
Le installatores e le pacchettos nunc installa un pauc de documentation in /usr/doc/slint-<version>-<date>, includente un Slint-HOWTO.txt e un script reinstall-slackware-packages.sh. +
Instructiones pro seliger le linguage de le installation nunc es fornite directemente in le schermo de benvenite de le installatores.

|19/12/2014|
Svedese nunc es disponibile, gratias a Ingemar Karlsson.
Gente cec nunc pote usar installatores de Slint installers (version 14.1) sin adjuta per usar un terminal Braille, gratias a Philippe Delavalade pro essayar e proponer como meliorar. +
Le software brltty que permitte usar un terminal Braille durante le installation es anque expedite e configurate in le systema installate, que assi pote esser executate completemente e gerite ex un terminal Braille. +
Le file de configuration de programma de dialogo e le commandos de dialogo que monstra un menu o un lista de selection ha essite modificate, principalmente pro facilitar le uso de un terminal Braille e anque pro personalisar le aspecto del quadratos de dialogo. +
Multe traductiones de files in le installatores, paginas de man e instrumentos administrative  ha essite addite, examinate e corrigite, e le pacchettos de Slint ha essite reconstruite in consequentia.
|========
