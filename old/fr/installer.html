L’installateur Slint

L’installateur Slint vous permet d’utiliser votre langue préférée durant l’installation, et ajoute des outils d’administration polyglottes et des pages de manuels traduites à Slackware.

Un terminal Braille peut aussi être utilisé pour installer, puis piloter le système. Un mode d’emploi est disponible ici: https://slint.fr/docs/README_BRLTTY

Télécharger et vérifier l’image ISO de l’installateur

Deux tailles d’images ISO sont disponibles :

  • Les images ISO DVD + USB complètes contiennent tout ce dont vous avez besoin pour installer Slackware.

  • Les petites images ISO CD + USB ont besoin en complément d’un média contenant les paquets Slackware, par exemple un DVD Slackware.

Pour télécharger une image ISO complète, cliquez d’abord sur son numéro de version, par exemple 14.1

Vous pouvez vérifier l’intégrité de l’image ISO avec sa somme de contrôle md5.

  • Pour les petites images ISO, en téléchageant et en utilisant le fichier md5 associé (dont le nom se termine par .md5).

  • Pour l’image ISO complète, vous pouvez afficher la somme de contrôle en cliquant sur la lettre i dans le disque noir à droite du lien vers l’image ISO. Par exemple, pour obtenir un installateur polyglotte d’une Slackware 64-14.1 internationalisée, téléchargez l’image ISO (et le fichier md5, dans le cas d’un petit installateur), puis dans le même répertoire tapez par exemple l’une des commandes suivantes : 

md5sum -c slint64-14.1.iso.md5 # le résultat devrait être : OK
md5sum slint64-14.1-full.iso # le résultat devrait être le même que celui affiché sur le site Internet

Graver l’image ISO sur un DVD, un CD ou une clef USB

Faire une clef USB amorçable

  • Tous le contenu précédent de la clef USB sera PERDU

  • Relisez soigneusement votre commande pour vous assurer de n’avoir pas écrit le nom d’une partition de disque dur au lieu de celui de la clef USB car le contenu de cette partition serait IRRÉMÉDIABLEMENT PERDU

Sous Linux insérez la clef USB, puis vérifiez son nom grâce aux commandes suivantes :

cat /proc/partitions
dmesg | tail

Si par exemple son nom est /dev/sdc, tapez :

dd if=<nomdefichier.iso> of=/dev/sdc bs=1M

Sous Windows utilisez l’application Rufus , gratuite et à code source ouvert.

faire un CD bootable (petit installateur) ou un DVD (grand installateur)

Sous Linux tapez une commande comme la première ci-dessous pour un CD, ou la seconde pour un DVD.

cat slint64-14.1.iso | cdrecord -v dev=/dev/cdrw speed=10 fs=8m -tao -eject -data -
growisofs -speed=2 -dvd-compat -Z /dev/sr0=slint64-14.1-full.iso

Sous Windows 2000/XP/Vista/7 utilisez l’application InfraRecorder, gratuite et à code source ouvert.

Sous Windows 7/8 utilisez l’application Graveur d’Image Disque Windows, livrée avec Windows 7 et 8

Utilisez votre installateur polyglotte

Insérez simplement le support d’installation (CD, DVD ou clef USB) et redémarrez votre machine.

Sur l'écran d’accueil, vous choisirez votre langue préférée pour l’installation en tapant le code à deux chiffres correspondant.

En plus d'être polyglotte, les installateurs Slint ajoutent les fonctionnalités suivantes à celles des installateurs Slackware d’origine:

  • Police par défaut et plan de clavier sont adaptés à la langue utilisée pour l’installation.

  • Quand c’est possible une police True Type est utilisée durant l’installation, pour adapter automatiquement la taille de police à la résolution d'écran (pour Slackware versions 14.0 et 14.1 seulement).

  • Il vous est proposé de créer des comptes d’utilisateurs ordinaires à la fin de l’installation (avant redémarrage).

  • Il vous est proposé de définir comme valeurs par défaut dans le système installé la langue et le plan de clavier utilisés durant l’installation, et d’effectuer des réglages correspondants du clavier sous X, si le serveur X est installé.

  • Un paquet Slint est installé, qui procure des traductions des principaux outils Slackware et de leurs pages de manuels, listées dans Paquet Slint, et des documents placés dans /usr/doc/slint-<version>-<date>, y compris un Slint-HOWTO.txt et un script reinstall-slackware-packages.sh.

Si vous utilisez une petite version de l’installateur, vous aurez aussi besoin durant l’installation d’un média source, autrement dit un média contenant les paquets Slackware. Les paquets peuvent être installés à partir de DVD/CDs Slackware, d’une clef USB Slackware, d’un partage Samba, de NFS (Network File System), d’un serveur FTP/HTTP. Quand vous choisissez SOURCE dans le menu, l’installateur vous dit quoi faire selon le type de média choisi.

Pour vous procurer un support source, suivez ce lien : Obtenir Slack

au besoin vous pouvez remplacer le média d’installation par un autre contenant les paquets, car le moment venu d’installer les paquets l’installateur aura déjà été chargé en mémoire.
