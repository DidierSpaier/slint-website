
= Пакунок Slint

Пакунок Slint містить багатомовні інструменти адміністратора та сторінки
керівництв, а також файли перекладів, надані проєктом Slint.

Ця сторінка пояснює, як отримати та встановити пакунок, а також дає опис
його вмісту, у тому числі, що та якими мовами перекладено.

До пакунку Slint поступово додаватиметься новий вміст, простим оновленням
пакунку ви зможете отримувати нові переклади інструментів і текстів, як і
додаткові мови, у міру їх випуску.

== Завантажити, встановити чи оновити пакунок Slint

НОТАТКА: Якщо ви використовували встановник Slint, додатково встановлювати
пакунок Slint не потрібно, це вже зроблено інсталятором.

Пакунок Slint надано для кожного підтримуваного випуску Slackware (Slackware
13.37, Slackware 14.0 та Slackware 14.1) у каталозі
https://slint.fr/packages[packages/]. Пакунок підходить як для 32-розрядної,
так і для 64-розрядної редакції. Щоб отримати, перевірити та встановити (або
оновити) пакунок Slint (наприклад, для Slackware 14.1), виконайте таке:

----
wget -r -l1 -np -nd -A slint-14.1*  https://slint.fr/packages/ # отримайте найновіший пакунок Slint та файл md5
md5sum -c slint-14.1*md5 # відповідь повинна бути позитивною, інакше завантажений пакунок пошкоджено
su # отримайте права root-а
upgradepkg --install-new --reinstall slint-14.1*txz # ця команда встановить або оновить пакунок
----

TIP: Аби 'slackpkg' не вилучав 'пакунок Slint', додайте до
/etc/slackpkg/blacklist рядок, що містить: + slint

== Що перекладено, якими мовами?

Пакунок Slint містить багатомовні інструменти та файли перекладу.

На дату складання цього документу включено такі мови: українська, російська,
французька, німецька, польська, портуґальська (Бразилія) та іспанська
(Латинська Америка).

Деякі сторінки керівництв також перекладено іншими мовами, їхній перелік
наведено у https://slint.fr/packages/MAN_PAGES.TXT[MAN_PAGES.TXT].

У таблиці нижче наведено багатомовні інструменти адміністратора та їх
призначення.

[format="psv", options="header,autowidth", style="verse"]
|========
|Багатомовний інструмент|Призначення|Перекладена сторінка керівництва
|adduser|Додає користувача до системи|
|explodepkg|Витягує вміст пакунку Slackware до поточного каталоґу|man explodepkg
|installpkg|Встановлює пакунки Slackware|man installpkg
|makebootdisk|Створює завантажувальну дискету Linux
(лише у Slackware 13.37 та 14.0)|
|makepkg|Створює пакунки Slackware|man makepkg
|pkgtool|Обслуговує пакунки проґрамного забезпечення та виконує завдання налаштування:
=> обрати сервіси для запуску при завантаженні
=> створити завантажувальний USB-носій
=> обрати тип мишки|man pkgtool
|removepkg|Вилучає пакунки Slackware|man removepkg
|upgradepkg|Оновлює пакунки Slackware|man upgradepkg
|eliloconfig|Налаштовує завантажувач 'elilo'
(лише у Slackware 14.1)|
|liloconfig|Налаштовує завантажувач 'lilo'|
|netconfig|Налаштовує мережу|
|timeconfig|Обирає часову зону|
|setconsolefont|Встановлює шрифт консолі|
|setup.mouse|Обирає тип мишки|
|xorgsetup|Налаштовує X сервер|
xwmconfig|Обирає типовий віконний менеджер або стільницю|
|slackpkg|Автоматизовані інструменти для керування пакунками Slackware
(перекладено лише сторінки керівництв)|man slackpkg
man slackpkg.conf
|========
 
NOTE: Аби бачити повідомлення багатомовних інструментів та сторінки керівництв
Вашою мовою, Ви повинні, як показано на домашній сторінці Slint,
link:slint.html#anchor-1[встановити бажану локаль]

Крім того, пакунок Slint встановлює документацію у директорію
/usr/docs/slint-<version>-<date>, включаючи Slint-HOWTO.txt та файл сценарію
reinstall-slackware-packages.sh.

TIP: за для переліку того, що встановив пакунок Slint, наберіть у терміналі:
----
less /var/log/packages/slint*
----
Для навігації по списку використовуйте стрілки вгору та вниз, щоб вийти
натисніть букву 'q' .
