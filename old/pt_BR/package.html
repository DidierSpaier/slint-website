Pacote Slint

O pacote Slint contém ferramentas de administração poliglotas e páginas de manual, bem como arquivos traduzidos, fornecidos pelo projeto Slint.

Esta página explica como obter e instalá-lo, em seguida, descreve o seu conteúdo, de modo que você sabe o que é traduzido e em que línguas.

Âmbito do pacote Slint irá expandir gradualmente, de modo que apenas atualizações lhe permitirão obter ferramentas e traduções atualizadas e recentemente traduzidas em mais línguas assim que elas estiverem disponíveis.

Transfira instala ou atualiza o pacote do Slint

Se você acabou de usar o instalador do Slint, não há necessidade de instalar o pacote Slint: visto já ter sido efetuado pelo instalador.

Um pacote Slint é fornecido para cada edição Slackware suportado (Slackware 13.37, Slackware 14.0 e Slackware 14.1) no diretório https://slint.fr/packages [pacotes /]. O mesmo pacote pode ser instalado em versões 32 bits e 64 bits. Para obter, verificar e instalar (ou atualizar) "pacote Slint " (para Slackware 14.1, neste exemplo), faça o seguinte:

wget -r -l1 -np -nd -A slint-14.1*  https://slint.fr/packages/ # você terá o mais recente pacote de Slint e seu arquivo md5 associado
md5sum -c slint-14.1*md5 # a resposta deve ser "OK", caso contrário o pacote baixado está corrompido
su # para ter privilégio de administrador
upgradepkg --install-new --reinstall slint-14.1*txz # este comando instala ou atualiza o pacote
Para evitar que o slackpkg remova o "pacote Slint ', adicione uma linha ao /etc/slackpkg /blacklist apenas com esta palavra: + slint

O que foi traduzido, em qual língua??

Um pacote Slint contém ferramentas poliglotas e arquivos traduzidos.

No momento da escrita, os seguintes idiomas estão incluídos: Francês, Alemão, Polonês, Português (Brasil), russo, espanhol (América Latina) e ucraniano.

Algumas páginas do manual foram traduzidas para outra língua, estão indicadas em MAN_PAGES.TXT.

A tabela a seguir lista as ferramentas de administração poliglotas e seus propósitos.

Ferramentas Poliglotas Propósitos Man pages traduzidas
adduser
Adicionar um usuário ao sistema
explodepkg
Extrair conteúdo de um pacote Slackware no diretório corrente
man explodepkg
installpkg
Instalar pacote Slackware
man installpkg
makebootdisk
Criar um disco de inicialização (somente nos Slackware 13.37 e 14.0)
makepkg
Crias pacotes para o Slackware
man makepkg
pkgtool
Manter pacotes de software e executar tarefas de configuração: ⇒ definir serviços para serem executados durante a inicialização ⇒ criar um dispositivo USB bootável ⇒ definir o modelo de mouse
man pkgtool
removepkg
Remover pacotes do Slackware
man removepkg
upgradepkg
Atualizar pacotes do Slackware
man upgradepkg
eliloconfig
Configura elilo carregador de boot (somente no Slackware 14.1)
liloconfig
Configura lilo carregador de boot
netconfig
Configura a rede
timeconfig
Define o fuso horário
setconsolefont
Define uma fonte
setup.mouse
Define o modelo do mouse
xorgsetup
Configura o servidor X
xwmconfig
Define o gerente de janelas padrão ou desktop
slackpkg
Ferramentas automatizadas para gerenciamento de pacotes do Slackware (apenas páginas de manual estão traduzidas)
man slackpkg man slackpkg.conf
Para ver as mensagens dos instrumentos poliglotas e páginas de manual no seu idioma, você deve, conforme indicado na home page Slint, link: slint.html # anchor-1 [configurar o locale preferido]

Além disso, o pacote Slint também instala documentos na pasta /usr/docs/slint-<versão>-<data>, incluindo um Slint-HOWTO.txt e um script reinstalar-slackware-packages.sh.

para mostrar o que o pacote Slint instalou, basta digitar num terminal: 
less /var/log/packages/slint*

Use as setas para cima e para baixo para navegar na lista, pressione a letra q para sair.
