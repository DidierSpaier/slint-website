Установщик Slint

Установщик Slint позволяет использовать при установке предпочитаемый язык и добавляет в Slackware переведённые инструменты администратора и страницы руководств.

Терминал Брайля может быть использован для установки и управления системой. Инструкции по использованию доступны в https://slint.fr/docs/README_BRLTTY.

Загрузка и проверка образа ISO установщика

Доступны образы ISO двух размеров

  • Полные DVD + USB образы ISO

  • Компактные CD + USB образы ISO, требующие носитель с пакетами Slackware, например, DVD Slackware.

Для загрузки полного образа ISO сперва щёлкните по номеру версии, например, 14.1.

Проверить целостность образа ISO возможно по его контрольной сумме md5.

  • Для компактных образов ISO используйте соответствующий файл md5 (его имя оканчивается на .md5).

  • Для полных образов ISO можно отобразить контрольную сумму щелчком по букве i в чёрном круге справа от ссылки на образ ISO. Например, чтобы сделать многоязычный установщик интернационализированной Slackware64-14.1, загрузите образ ISO (для компактного установщика ещё и файл md5), затем в том же каталоге выполните одну из следующих команд: 

md5sum -c slint64-14.1.iso.md5 # результат должен быть положительным
md5sum slint64-14.1-full.iso # результат должен быть как на веб-сайте

Запись ISO-образа на DVD, CD или USB-носитель

Создание загрузочного USB-носителя

  • Все предыдущее содержимое USB-носителя будет УДАЛЕНО

  • Перепроверьте вашу команду, чтобы быть уверенным, что не вписали вместо USB-носителя имя раздела жесткого диска, поскольку содержимое этого раздела будет УДАЛЕНО БЕЗВОЗВРАТНО

В Linux подключите USB-носитель, затем узнайте его имя следующей командой:

cat /proc/partitions
dmesg | tail

Например, если его имя /dev/sdc, наберите:

dd if=<filename.iso> of=/dev/sdc bs=1M

В Windows воспользуйтесь приложением Rufus, оно бесплатно и его исходный код открыт.

Создание загрузочного CD (компактного установщика) или DVD (полного установщика)

В Linux выполните одну из следующих команд, для CD первую, для DVD - вторую.

cat slint64-14.1.iso | cdrecord -v dev=/dev/cdrw speed=10 fs=8m -tao -eject -data -
growisofs -speed=2 -dvd-compat -Z /dev/sr0=slint64-14.1-full.iso

В Windows 2000/XP/Vista/7 воспользуйтесь приложением InfraRecorder, оно бесплатно и его исходный код открыт.

В Windows 7/8 воспользуйтесь Windows Disk Image Burner, поставляемым вместе с Windows 7 и 8

Использование многоязычного установщика

Просто вставьте установочный носитель (CD, DVD или USB) и перезагрузите машину.

На экране приветствия выберите язык установки, введя соответствующий двухсимвольный код.

Многоязычные установщики Slint помимо возможностей оригинального установщика Slackware предлагают следующее:

  • Шрифт и раскладка клавиатуры адаптированы к языку установки.

  • Когда возможно, для адаптации к разрешению экрана при установке используется шрифт True Type (только Slackware версий 14.0 и 14.1).

  • Возможно создать обычного пользователя по завершении установки (до перезагрузки).

  • Возможно задать для установленной системы язык и раскладку клавиатуры, использовавшиеся при установке, и соответственно настроить клавиатуру в X, если был установлен сервер X.

  • Устанавливает пакет Sint с переводами перечисленных по ссылке :package.html[Пакет Slint] основных утилит Slackware и их страниц руководств, а также документацией в каталоге /usr/doc/slint-<версия>-<дата>, включая Slint-HOWTO.txt и сценарий reinstall-slackware-packages.sh.

Если для установки используете компактную версию установщика, потребуется и исходный носитель, то есть, носитель, содержащий пакеты Slackware. Пакеты могут быть установлены с CD или DVD Slackware, USB-носителя Slackware, из предварительно смонтированного каталога, с раздела жесткого диска, из сетевого каталога Samba, с NFS (Network File System, Сетевая Файловая Система), с сервера FTP/HTTP. После выбора в меню пункта ИСТОЧНИК, установщик подскажет, что делать дальше для выбранного типа носителя .

Чтобы получить носитель-источник, перейдите по ссылке: Получить Slack

При необходимости можно заменить установочный носитель тем, что содержит пакеты; к тому моменту установщик уже будет полностью загружен в оперативную память.
