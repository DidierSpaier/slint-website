Встановник Slint

Встановник Slint під час встановлення дозволяє використовувати бажану мову та додає до Slackware перекладені інструменти адміністратора і сторінки керівництв.

Термінал Брайля може бути використаний для встановлення і керування системою. Інструкції з використання доступні в https://slint.fr/docs/README_BRLTTY.

Завантаження та перевірка ISO-образу інсталятора

Доступні ISO образи двух розмірів:

  • DVD + USB повні ISO образи включають в себе, усе що треба, для встановлення Slackware.

  • CD + USB невеликі ISO образи потрібні, для доповнення носія з пакунками Slackware, наприклад для Slackware DVD.

Для завантаження повного образу ISO, спочатку клацніть по йього версії, наприклад 14.1

Ви можете перевірити цілістніть образу ISO, за допомогою контрольною суми md5

  • Для невеличких образів ISO, завантажте та використайте асоційований md5 файл (чиє ім’я закінчується на .md5).

  • Для повного ISO образа, Ви можете відобразити контрольну суму, натиснувши на букву у чорному колі, праворуч від посилання на ISO образ. Наприклад, щоб зробити багатомовний встановник з інтернаціоналізованою Slackware64-14.1, завантажте ISO образ (та md5 файл, у випадку використання маленького образа) потім в тій самій директорії, введіть наприклад одну із слідуючих команд 

md5sum -c slint64-14.1.iso.md5 # результат повинен бути позитивним
md5sum slint64-14.1-full.iso # результат повинен бути таким самим, як показано на сайті

Запис ISO-образу на DVD, CD або на USB-носій

Створення завантажувального USB-носія

  • Увесь попередній вміст USB-носія буде ВИЛУЧЕНО

  • Ще раз перевірте Вашу команду, аби бути впевненим, що не вписали замість USB-носія ім’я розділу жорсткого диска, оскільки вміст цього розділу буде ВИЛУЧЕНО НАЗАВЖДИ

У Linux під’єднайте USB-носій, потім дізнайтеся його ім'я такою командою:

cat /proc/partitions
dmesg | tail

Наприклад, якщо його ім’я /dev/sdc, наберіть:

dd if=<filename.iso> of=/dev/sdc bs=1M

У Windows скористайтеся застосунком Rufus, він безкоштовний і його джерельний код відкритий.

Створити завантажувальний CD (маленький встановник) чи DVD (великий встановник)

На Linux наберіть команду нижче, першу для CD чи другу для DVD.

cat slint64-14.1.iso | cdrecord -v dev=/dev/cdrw speed=10 fs=8m -tao -eject -data -
growisofs -speed=2 -dvd-compat -Z /dev/sr0=slint64-14.1-full.iso

У Windows 2000/XP/Vista/7 скористайтеся застосунком InfraRecorder, він безкоштовний і його джерельний код відкритий.

У Windows 7/8 скористайтеся Windows Disk Image Burner, що постачається разом із Windows 7 та 8

Використовуйте ваш багатомовний встановник

Просто вставте встановлювальний носій (CD, DVD або USB) та перезавантажте машину.

На екрані привітання, Ви можете вибрати мову для встановлення, наберіть відповідні дві цифри.

Крім того багатомовний встановник Slint, пропонує, додаткові можливості, до офіційних встановника Slackware:

  • Шрифт на розкладка клавіатури за замовченням, пристусовується під обрану мову встановлення.

  • Якщо можливо, шрифт True Type використовується під час встановлення, для автоматичного пристосування розміру шрифту під роздільну здатність екрану (тількі для весії Slackware 14.0 та 14.1).

  • Забезпечено можливість, створення облікових записів звичайних користвачів, в кінці встановлення (перед перезавантаженням).

  • Забезпечено можливість встановити обрану мову та розкладку клавіатури під час встановлення, як налаштування за замовченням, а також, налаштування клавіатури для X серверу.

  • Пакунок Slint встановленно, що забезпечує переклади основних інструментів Slackware та їх керівництва, перерахованих у Slint, також документація розміщена у директорії /usr/doc/slint-<version>-<date>, включаючи Slint-HOWTO.txt та файл сценарію reinstall-slackware-packages.sh.

Якщо ви використовуєте маленьку версію встановника, вам також треба джерельний носії, іншими словами, носій в якому є пакунки Slackware. Пакунки можуть бути встановлені з носіїв DVD/CD/USB, примонтованої директорії, розділу жорсткого диску, спільного ресурсу SAMBA, NFS (мережева файлова система) чи з FTP/HTTP серверу. Коли ви оберете ДЖЕРЕЛО в меню, встановник Вам підкаже. що робити з обраним носієм.

Щоб отримати носій-джерело, перейдіть за посиланням: Отримати Slack

Якщо треба, Ви можете замінити встановлювальний носій, іншим (який містіть наступні пакунки), станом на момент установки пакетів, встановник вже завантажен у пам’ять.
